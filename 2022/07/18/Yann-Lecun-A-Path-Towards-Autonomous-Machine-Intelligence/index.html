Yann Lecun: A Path Towards Autonomous Machine Intelligence

发表于2022-07-18|更新于2022-07-19|文献阅读
|字数总计:2.7k|阅读时长:7分钟

摘要

机器怎样才能像人和动物一样高效地学习？机器怎样学习推理和规划？机器怎样在多种抽象的层次上学习感知和动作规划的表示，去推理、预测并在多种时间维度规划本文提出构建自主智能体的架构和训练范式。它结合了例如可配置的预测世界模型、通过内在动机驱动的行为以及通过自监督学习训练的分层联合嵌入架构。

关键词： 人工智能 机器常识 认知架构 深度学习 自监督学习 基于能量的模型 世界模型 联合嵌入架构 内在动机

引言

动物和人表现出学习和理解世界的能力远超目前人工智能和机器学习的能力。

一个青少年怎么可能在大约20个小时的练习中学会开车，孩子们怎么可能通过少量的接触就学会语言？为什么大多数人会知道如何在许多他们从未遇到过的情况下采取行动?相比之下，为了更可靠，目前的机器学习系统需要进行大量的试验训练，这样即使是最罕见的情况组合也会在训练中经常遇到。尽管如此，即使在接受了人类专家提供的大量监督数据后，在虚拟环境中经历了数百万次强化学习试验后，在工程师将数百种行为硬连接到系统中之后，我们最好的机器学习系统在驾驶等现实世界任务中的可靠性仍远不及人类。

答案可能在于人类和许多动物学习世界模型的能力，世界如何运作的内部模型。

目前，人工智能研究必须应对三个主要挑战:

  1. 机器如何学习描述世界，学习预测，学习主要通过观察来行动？
    现实世界中的互动既昂贵又危险，智能体应该在没有互动的情况下(通过观察)尽可能多地了解这个世界，从而尽可能减少学习一项特定任务所需的昂贵而危险的试验次数。

  2. 机器如何能够以与基于梯度的学习兼容的方式进行推理和规划？
    我们最好的学习方法依赖于估计和使用损失的梯度，这只能用可微分的架构来执行，并且很难与基于逻辑的符号推理相协调。

  3. 机器如何学习以一种分层的方式，在多个抽象级别和多个时间尺度上表示感知和行动计划？
    人类和许多动物都能够构思出多层次的抽象概念，通过将复杂的行动分解成一系列较低层次的行动，可以实现长期的预测和长期的规划。

本文提出了一种智能体的架构，为所有这三个挑战提供了可能的解决方案。

本文的主要贡献如下:

  1. 一个整体的认知架构，其中所有模块都是可区分的，其中许多模块是可训练的（图2，第3节）。
  2. JEPA和分层JEPA：预测世界模型的非生成架构，学习了表示的层次结构（章节4.4和4.6，图12和15）
  3. 一种非对比的自我监督学习范式，产生的表征同时是信息量大和可预测的（节4.5，图13）。
  4. 一种在不确定性下，使用H-JEPA作为分层规划预测世界模型的基础的方法（第4.7节，图16和17）。

学习世界模型

人类和非人类的动物似乎能通过观察和不可理解的少量互动，以一种任务独立、无监督的方式，学习大量关于世界如何运作的背景知识。可以假设，这些积累的知识也许组成了我们常说的常识的基础。常识可以视为世界模型的集合，可以告诉智能体什么是可能的，什么是可信的，以及什么是不可能的。用这样的世界模型，动物可以通过非常少的试验学习新的技能。它们可以预测动作的结果，它们可以推理，规划，探索，以及想象问题的新解决方案。重要的是，当面对一个未知情况时，它们也可以避免犯危险的错误。

人类，动物和智能系统使用世界模型的想法可以追溯到很久以前的心理学。自20世纪50年代以来，使用预测世界下一个状态作为当前状态和被考虑的行动的函数的正演模型，一直是最优控制的标准程序，被称为模型预测控制。强化学习中使用可微世界模型已经被长期忽视，但正在卷土重来。

汽车的自动驾驶系统也许需要数千次强化学习的试验，以了解转弯时开得太快会造成不好的结果，并学会减速以避免打滑。对比之下，人类可以利用他们对直观物理的熟悉知识来预测这样的结果，并且在学习新技能时很大程度上避免致命的行动过程。

无论是时间上的还是空间上的，常识不仅允许动物预测未来的结果，而且能填补缺失的信息。这允许它们对产生与常识一致的感知解释。当面对一个模棱两可的感知时，常识允许动物不去理会与他们内部世界模型不一致的解释，并给予特别关注，因为它可能表明一个危险的情况和学习一个精致的世界模型的机会。

我认为，设计学习范式和架构，使机器能以无监督（或自监督）的方式学习世界模型，并用这些模型去预测、推理和规划，是当今人工智能和机器学习的主要挑战之一。一个主要的技术障碍是如何设计可训练的世界模型，以处理预测中的不确定性。

人类和动物学习模型的层次

人类和非人类的动物在生命的最初几天、几周和几个月里学习世界怎样运作的基本知识。尽管我们很快就可以获得大量这样的知识，但这些知识似乎是如此基础，以至于我们认为理所当然。

在生命的最初几个月，我们知道世界是三维的。我们知道每一种光源、声音和触摸都与我们有一定的距离。视觉感知中的每个点都有一定的距离，这个事实最好地解释了我们从左眼到右眼或者当我们头部移动时世界时如何变化的。运动视差使深度变得明显，从而使物体的概念变得明显，以及物体可以遮挡更远的物体的事实。一旦确定了物体的存在性，它们可以根据其外观或行为被自动划分为更广的类别。

在物体的概念之上，我们知道物体不会自发地出现、消失、改变形状或者瞬间移动：它们会平稳地移动，并且任何时间只能出现在一个地方。一旦掌握了这些概念，很容易知道，有些物体是静止的，有些物体有可预测的轨迹（无生命的物体），有些物体的行为有些不可预测（集体现象，如水、沙、风中的树叶等），有些物体看起来遵循不同的规则（有生命的物体）。直观物理的概念，如稳定性、重力、惯性和其他等，可以再此基础上出现。有生命的物体对世界的影响（包括主体自身行为的影响）可以用于推断因果关系，在此基础上可以获得语言和社会知识。

图一展示了婴儿在什么年龄获得基本概念，如物体持久性、基础类别和直观物理等。高层次的抽象概念似乎是在低层次的概念上发展而来的。

有了这种世界的认知，再结合简单的固有行为和内在动机/目标，动物可以很快学习新任务，预测它们行动的结果并提前规划，预见成功的行为路线并避免危险的情况。

但是人类和动物大脑可以包含生存所需要的所有世界模型吗？本文中的一个假设是动物和人类的前额皮质中只有一个世界模型引擎。这个世界模型引擎可以为手头任务动态配置的。使用这个单一的、可配置的世界模型引擎，而不是对每种情况都有一个单独的模型，关于世界如何运作的知识可以跨任务共享。这可以通过将一种情况的配置模型应用到另一种情况来进行类比推理。
